# Демо-проект: каталог + слайдер

## Суть
Создать демонстрационный веб-проект с каталогом и интерактивным слайдером на базе Embla Carousel.

## Ключевые особенности
- **Слайдер**: Embla Carousel с плавными переходами, touch-свайпами, автопроигрыванием
- **Визуал**: Эффект glassmorphism (стеклянные карточки, полупрозрачные overlays)
- **Переходы**: Плавные CSS-анимации и transform для 60fps
- **Кроссбраузерность**: Поддержка всех современных браузеров с graceful degradation
- **Производительность**: Lazy loading, аппаратно-ускоренные анимации, оптимизированный JS

## Цель
Демонстрация высокого уровня фронтенд-разработки: красивый UI + отличная производительность.
