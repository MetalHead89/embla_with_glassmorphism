## ✅ Задачи

- [x] **1. Инициализация проекта**
  - `nuxi init` + TypeScript + Tailwind
  - Настройка `nuxt.config.ts` + `tailwind.config.js`
  - Структура папок: `components/`, `types/`, `pages/`
  - **Тест**: `npm run dev` → http://localhost:3000

- [x] **2. Базовая верстка + тема**
  - Dark mode фон страницы
  - Tailwind база + SCSS модули
  - Глобальные стили + glassmorphism переменные
  - **Тест**: Открыть сайт — проверить фон/темы

- [x] **3. Hero-секция (фонд категории)**
  - Компонент `HeroSection.vue`
  - Фоновое изображение (cover)
  - Заглушка при ошибке загрузки
  - **Тест**: Отображение херо с фоном

- [x] **4. Слайдер категорий (Embla)**
  - Компонент `CategorySlider.vue`
  - Чипсы категорий + glassmorphism
  - Активный чипс (scale 1.05)
  - Touch-свайпы + автоплей (пауза при ховере)
  - **Тест**: Свип/клик → смена активной категории

- [x] **5. Сетка карточек товаров**
  - Компоненты `ProductGrid.vue` + `ProductCard.vue`
  - Адаптивность: 1/2/4 колонки
  - Glassmorphism карточек + fallback изображений
  - Lazy loading изображений
  - **Тест**: Рендер 50 карточек без лагов

- [ ] **6. API-связка (DummyJSON)**
  - `types/index.ts` — интерфейсы Product/Category
  - `useFetch` загрузка категорий + товаров первой категории
  - Кэш в памяти на сессию
  - Обрезка до 50 товаров
  - **Тест**: Клик по категории → обновление сетки

- [ ] **7. Анимации переходов**
  - Херо: cross-fade 0.4s ease
  - Сетка: stagger fade-in (50ms delay, 0.3s duration)
  - Трансформации через CSS (transform, translate3d)
  - **Тест**: Смена категории → плавные анимации

- [ ] **8. Оптимизация + Lighthouse**
  - Tree-shaking (Vite/Nuxt)
  - `will-change` для анимируемых элементов
  - Фиксация размеров (CLS < 0.1)
  - Запуск Lighthouse — проверка Performance ≥90
  - **Тест**: Lighthouse отчет — все Web Vitals в цели

- [ ] **9. Футер + финальный штрих**
  - Компонент `Footer.vue`
  - Текстовый контент
  - Отладка responsive на всех экранах
  - **Тест**: Полноэкранное тестирование (моб/планшет/десктоп)
